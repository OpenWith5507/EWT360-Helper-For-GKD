# EWT360-Helper-For-GKD
升学e网通的GKD click scripts
## 已经实现的功能
- 自动进入任务界面
- 自动开始看课
- 自动点认真度检查
- 自动连播
## 注意事项
**本项目仅供研究学习，请在24小时内删除。使用该订阅可能产生的后果应由使用者承担**
- 需要自己开倍速
- 可能会增加耗电
- 广告弹窗可能要手动关闭
- 二改发布请标明原作者
- 请不要发Q群
