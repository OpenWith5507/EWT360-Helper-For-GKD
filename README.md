# EWT360-Helper-For-GKD
基于GKD的升学e网通自动点击脚本订阅
## 已经实现的功能
- 自动进入任务界面
- 自动开始看课
- 自动点认真度检查
- 自动连播
- 自动关闭眼保健操打断（40分钟连续看时会出现）
## 注意事项
**本项目仅供研究学习，请在24小时内删除。使用该订阅可能产生的后果应由使用者承担**
- 需要自己开倍速
- 可能会增加耗电
- 广告弹窗可能要手动关闭
- 二改发布请标明原作者
- 禁止用于盈利性活动
## 使用方法
1. 下载GKD并开启无障碍服务
https://github.com/gkd-kit/gkd/releases/
2. 复制下方链接导入订阅
https://raw.githubusercontent.com/OpenWith5507/EWT360-Helper-For-GKD/main/ewt.json5
备用链接https://git.silica.codes/OpenWith/test/raw/branch/main/ewt.json5
